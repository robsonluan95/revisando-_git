## Dominando Branch

É uma meneira de que o git disponibiliza para separar as versões do projetos

Quando criamos um repositorio ele sempre inicia na Branch main - quando começamos a programar nunca criamos dietamente na main, ja que esse codigo é a principal. Devemos criar uma segundaria para caso tenha erros e depois unimos

## Como criar uma Branch

git branch -M main → ele volta pra o jeito que estar no github mas o /main é o nome da branch
