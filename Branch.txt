## Dominando Branch

É uma maneira de que o git disponibiliza para separar as versões do projetos

Quando criamos um repositório ele sempre inicia na Branch main - quando começamos a programar nunca criamos dietamente na main, ja que esse codigo é a principal. Devemos criar uma segundaria para caso tenha erros e depois unimos

## Como criar uma Branch

git branch page_branch → Para criamos uma branch (nosso exemplo vamos usar page_branch)

git branch → Para visualizar todas as branch

git checkout page_branch → Para mudar para branch que deseja(nesse caso o exemplo é page_branch)

git checkout -b “page_git_branch” → Para mudar automaticamente para branch que foi criada(nesse caso o exemplo é page_git_branch)

git branch -d page_git_branch → para deletar um branch ( -d → delete. Nesse caso o exemplo é page_git_branch)

## Unindo Branches

git merge page_branch → Para unirmos branch's (nosso exemplo vamos usar page_branch)
